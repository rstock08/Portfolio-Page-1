About Me

Having moved to San Diego in August of 2018, I have since been employed as a Junior Software Engineer at HII - Technical Solutions Division. As the only onsite Software Engineer, I wear many hats. Among my responsibilities , I act as a liaison between our customers, Software Engineers, and the IT team. Additionally, I provide QA testing along with detailed bug reports.

When I am wearing my Software Engineering hat I write and maintain a custom-built installation package both the main application as well as several prerequisite applications necessary to run it. I've also written CSV Exporters that use RabbitMQ to collect information from our PostgreSQL database. This information is then turned into an easily read CSV file which is used to improve our machine learning algorithm. These applications have saved hundreds of man-hours as the information fed to the algorithm was manually gathered before and each prerequisite software was manually installed - taking out as much of the human factor in repetitive tasks means fewer errors!

On my own, I have continued to advance as a Software Engineer. Recently I wrote this website. I learned how to use Bootstrap 4 to streamline the process of creating responsive websites. Additionally, I am learning more about Object Oriented Design on my own with sites set on learning JavaScript Frameworks such as Springboot and Angular.
